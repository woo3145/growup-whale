## 🛠️ Stack

### Backend
- 언어: `Python`
- 프레임워크: `Flask`
- DB: `SQL3LITE`
- ORM: `SQLALCHEMY`
- 인증: `JWT`
- 배포: `AWS-S3`, `PythonAnyWhere`

### Front
- 언어: `JavaScript`, `HTML`
- CSS: `Bootstrap`
- 템플릿 엔진: `Jinja2`
